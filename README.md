# time-cli
 Get the current Unix time in milliseconds or converts an epoch to human readable time.


## Installation
```sh
npm i -g time-cli
```

## Usage
### Get current Unix time
```sh
time-cli
# 1581490592713
```
### Covert
```sh
time-cli 1581490592713
# 2/12/2020, 2:56:32 PM
# 2 minutes ago
```
```sh
time-cli 12314312123412
# 3/24/2360, 2:15:23 AM
# 340 years from now
```
